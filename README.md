# Jet Clustering with `fastjet`

Short tutorial for getting started with some jet clustering using `fastjet` in python.

## Setup

This project uses [`uv`](https://docs.astral.sh/uv/) for environment and package management.

### Install `uv`

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### Create environment and install dependencies

```bash
uv sync
```

This creates a virtual environment in `.venv/` and installs all dependencies declared in `pyproject.toml`.

Activate the environment with

```bash
source .venv/bin/activate
```
